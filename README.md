# STUDGUARD
...is a server moderation system that includes a scripted moderator that will moderate your server. It also includes for-fun  commands such as /burn which burns a player or /listitems


Full list of commands:
(Note: All commands are togglable by SAs only via RTB prefs)

/listitems [playername] - Shows the items in a player's inventory | (Moderator+ only) (Changable via RTB Prefs)
/rules - Displays rules set via RTB prefs 
/list - Lists all players online along with their tags and ranks
/title - Gives a suffix to a player | (Admin+ Only)
/removesuffix - Removes your title

and probably more to be added
